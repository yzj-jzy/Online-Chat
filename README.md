# Configration
Make sure the file location always under ~/online-chat
1. Download the Vue and Dependency, lottie-web Module
```
npm install vue
npm install --force
npm install lottie-web
```

2. Run services and open 8080 [portal](http://localhost:8080/#/)  in browser
```
npm run serve
```
